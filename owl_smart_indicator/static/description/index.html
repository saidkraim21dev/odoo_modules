Bring Your Data to Life

Stop looking at boring static numbers. OWL Smart Status Indicator transforms any numeric field into an instant visual signal — green, amber, or red. Perfect for Stock Levels, Quality Ratings, Project Progress, or any Numeric KPI.

Drop it into any Form, Tree, or Kanban view with a single attribute. Zero configuration required.

<field name="progress_rate" widget="status_indicator"/>

How It Works

Three intelligent states — automatically determined from your field value.

≥ 80%

Optimal

Everything is on track. A clean success signal keeps attention where it's needed most.

40 – 79%

Warning

Attention is needed. Value is in a degraded range that requires timely review.

< 40%

Critical

Immediate action required. High-priority issues surface before they escalate.

Zero-Configuration Usage

Works in Form, Tree, and Kanban views. Simply add the widget attribute — nothing else.

XML · Odoo View
<!-- Form View -->
<field
    name="progress_rate"
    widget="status_indicator"/>

<!-- List View -->
<field
    name="quality_score"
    widget="status_indicator"/>

<!-- Kanban View -->
<field
    name="fill_rate"
    widget="status_indicator"/>
